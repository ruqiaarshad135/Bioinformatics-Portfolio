# In-silico Analysis of SALL4 Gene

## Objective
Perform basic bioinformatics analysis of the SALL4 gene to demonstrate basic computational biology skills.

## Tools Used
- NCBI BLAST
- ExPASy ProtParam

## Steps
1. Downloaded SALL4 gene/protein sequence from NCBI
2. Performed BLAST analysis to identify homologous sequences
3. Analyzed protein properties using ExPASy ProtParam
4. Documented results
